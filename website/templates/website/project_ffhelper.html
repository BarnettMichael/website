{% extends "website/project_template.html" %} {% load staticfiles %} {% block head_title %}Fantasy Football Helper Project{% endblock %} {% block content_title %} Fantasy Football Helper {% endblock %} {% block project_description %} Fantasy Football Helper helps you pick your NFL fantasy football team by providing recommended players based on their past performances, what positions you currently need to fill or whether you already have a full compliment of starters. The script recommends those players who are still available with the best historic performance and also those who have the best value in comparison to the mean of players in their position.

This project includes a web scraper written using the scrapy library, The data collected is stored in a postgres Database data is collected and parsed by assigning xpath locations into PlayerItem objects. Which are stored using a sqlalchemy pipeline into the database.
Data in the database is then retrieved via sending SQL queries when the app software is opened is used to display information to the user via the GUI {% endblock %} {% block project_skills %}
  • Python 2.7 - The webscraper, GUI and logic of the app is all written in python.
  • Scrapy + Xpath selectors - A python package that provides webscraping tools, used to write the spider that collected the data.
  • SQL Alchemy - Used as part of the pipeline to input data into the database, used to receive data from scrapy and input it into a database by creating and sending the raw SQL for inputting user defined objects.
  • PostGres Database - Used to store the data in a relational database.
  • psycopg2 - Used to connect to a database and send a cursor with user written raw SQL code to be executed.
  • TKinter GUI - GUI package that is part of the base python library.
