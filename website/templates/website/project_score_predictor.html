{% extends "website/project_template.html" %} {% load staticfiles %} {% block head_title %}Score Predictor Project{% endblock %} {% block content_title %} Score Predictor Challenge{% endblock %} {% block project_description %} The Score Predictor Challenge is a website that allows you to compete against other users to try and guess the results of sports matches. The website allows you to guess who will win upcoming sports matches and by what margin. User guesses are tracked and points are assigned based on whether or not the correct team was selected, with bonus points awarded to the user(s) who is closest to the actual winning margin. The homepage displays the current standings of users, a list of upcoming matches and links to make/edit user guesses for upcoming matches and a view of all previous results and the guesses each user made associated with them.

The website is made using Django, with data being stored in a MySql database. {% endblock %} {% block project_skills %}

Python 2.7 - All logic for the website is written in python.
HTML 5 - Used to write the Templates in a responsive design.
CSS - Custom CSS written to style elements in the templates to improve usability.
Javascript/Jquery - Used to create an interactive dropdown menu.
